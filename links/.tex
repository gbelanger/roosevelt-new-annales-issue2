                          LYNX - The Text Web-Browser

   Lynx is the text web browser.

   This is the toplevel page for the Lynx software distribution site.

   The [6]current development sources have the latest version of Lynx
   available (development towards 2.9.0).
   The [7]main help page for lynx-current is online; the current [8]User
   Guide is part of the online documentation.

   The most recent stable release is [9]lynx2.8.9.

   Other resources include:
     * [10]Mailing list archives
     * [11]pgp/gpg signatures
